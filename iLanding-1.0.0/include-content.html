How it works:

This demo uses JavaScript to dynamically insert HTML content. In a real application, you would fetch content from external files.

// Example fetch implementation:
fetch('content/header.html')
  .then(response => response.text())
  .then(data => {
    document.getElementById('content').innerHTML += data;
  });
