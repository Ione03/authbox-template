Closing Bootstrap Modals

Multiple ways to close a Bootstrap modal with examples

Methods to Close a Bootstrap Modal

1 Basic Modal Structure

First, let's create a basic Bootstrap modal:

<!-- Button to trigger modal -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  Open Modal
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1" aria-labelledby="exampleModalLabel" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title" id="exampleModalLabel">Modal Title</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Close"></button>
      </div>
      <div class="modal-body">
        Modal content goes here...
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
        <button type="button" class="btn btn-primary">Save changes</button>
      </div>
    </div>
  </div>
</div>

2 Methods to Close the Modal

1. Using data-bs-dismiss Attribute

Add data-bs-dismiss="modal" to any button inside the modal to close it when clicked.

<button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>

2. Using JavaScript hide() Method

Call the hide() method on the modal instance.

// Get the modal instance
var myModal = new bootstrap.Modal(document.getElementById('exampleModal'));
// Hide the modal
myModal.hide();

3. Using jQuery

If you're using jQuery, you can close the modal with:

// Hide the modal
$('#exampleModal').modal('hide');

4. Closing by Clicking Outside

By default, clicking outside the modal content will close it. You can disable this with data-bs-backdrop="static".

<!-- Modal with static backdrop (won't close when clicking outside) -->
<div class="modal fade" id="exampleModal" data-bs-backdrop="static" data-bs-keyboard="false">

5. Closing with Escape Key

By default, pressing the Escape key will close the modal. You can disable this with data-bs-keyboard="false".

<!-- Modal that won't close with Escape key -->
<div class="modal fade" id="exampleModal" data-bs-backdrop="true" data-bs-keyboard="false">

3 Advanced Example: Programmatic Control

Here's an example showing how to control modals programmatically with JavaScript:

// Get the modal element
var exampleModal = document.getElementById('exampleModal');

// Create a Bootstrap modal instance
var modal = new bootstrap.Modal(exampleModal);

// Show the modal
modal.show();

// Add event listener for when the modal is fully hidden
exampleModal.addEventListener('hidden.bs.modal', function (event) {
  // Do something when the modal is completely hidden
  console.log('Modal is now hidden');
});

// Hide the modal after 5 seconds
setTimeout(function() {
  modal.hide();
}, 5000);
