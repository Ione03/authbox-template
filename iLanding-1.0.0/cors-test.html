Solution Explanation

Why your request is being blocked:

Browser security prevents frontend JavaScript from making requests to a different domain/port than the one serving the page (CORS policy).

Solutions:

  1. Install django-cors-headers in your Django project
  2. Configure CORS in your settings.py
  3. Use a proxy during development
  4. Handle credentials properly

Configuration Example:

# settings.py
INSTALLED_APPS = [
    ...
    'corsheaders',
    ...
]

MIDDLEWARE = [
    ...
    'corsheaders.middleware.CorsMiddleware',
    'django.middleware.common.CommonMiddleware',
    ...
]

# For development only - be more restrictive in production
CORS_ALLOW_ALL_ORIGINS = True
# Or specify allowed origins:
# CORS_ALLOWED_ORIGINS = [
#     "http://localhost:3000",
#     "http://127.0.0.1:3000",
# ]
